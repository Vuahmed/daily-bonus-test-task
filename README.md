# daily-bonus-test-task

## Описание

Данный проект представляет собой реализацию системы ежедневного бонуса в Unity.

При запуске игры пользователь получает случайный ежедневный бонус.  
Система поддерживает два режима работы:
**Test (тестовый режим)** — ускоренное время (24 минуты)
**Release (релизный режим)** — стандартное реальное время (24 часа)

После получения бонуса запускается таймер до следующего доступного бонуса.

## Основной функционал

- Генерация случайного ежедневного бонуса при запуске
- UI контейнер для отображения текущего активного бонуса
- UI контейнер для отображения списка всех возможных бонусов
- Поддержка конфигурации через JSON файл
- Переключение режимов Test / Release
- Таймер обратного отсчета до следующего бонуса
- Отображение информации о текущем бонусе в UI

## Архитектура

BonusSystem    — основная логика выдачи и таймеров
BonusVisual    — UI слой (отображение данных)
BonusDatabase  — ScriptableObject база бонусов
ConfigLoader   — загрузка конфигурации (JSON)

## Использованные технологии

- Unity Engine
- ScriptableObjects
- PlayerPrefs (сохранение состояния)
- Coroutines (таймер)
- Events (Action)
- JSON конфигурация (Resources)
- Unity UI (TextMeshPro)

## Структура бонуса

Каждый бонус содержит:

- `string id` — уникальный идентификатор
- `Sprite avatarBonus` — иконка бонуса
- `float timeBonus` — длительность действия бонуса

Примеры бонусов:
- временное увеличение максимального здоровья
- усиление характеристик
